## Overview

Developed autonomous robotic systems capable of navigating and operating in unstructured field environments. This project integrated multiple sensor modalities including LiDAR, cameras, and IMUs for robust perception and navigation.

## Key Contributions

- Sensor fusion pipeline combining LiDAR and vision data
- Path planning algorithms for uneven terrain
- Real-time obstacle avoidance system
- Field testing and validation in outdoor environments
