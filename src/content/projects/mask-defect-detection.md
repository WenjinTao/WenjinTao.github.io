## Overview

Built an automated quality inspection system using deep learning to detect defects in face masks during high-speed production. The system runs in real-time on production lines.

## Key Contributions

- Custom CNN architecture for defect classification
- Data augmentation pipeline for limited training samples
- Real-time inference on edge devices
- Integration with production line PLC systems
