## Overview

Developed a high-precision visual inspection system for electronic pin components, achieving sub-millimeter defect detection accuracy using deep learning deployed on edge computing hardware.

## Key Contributions

- High-resolution imaging pipeline for micro-component inspection
- Transfer learning approach for small defect detection
- Edge deployment for low-latency inference
- Automated reporting and quality tracking dashboard
